# Grab Tool

Select entire shapes to move/flip or duplicate them

#### Controls:

| Controller | Input             | Result    |
| ---------- | ----------------- | --------- |
| Main       | Directional left  | Duplicate |
|            | Directional right | Flip      |
|            | Directional up    | Size +    |
|            | Directional down  | Size -    |
|            | Trigger           | Grab      |
