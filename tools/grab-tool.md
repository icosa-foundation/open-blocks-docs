# Grab Tool

Select entire shapes to move, scale, flip, group or duplicate them

#### Controls:

| Controller | Input             | Result                |
| ---------- | ----------------- | --------------------- |
| Main       | Directional left  | Duplicate             |
|            | Directional right | Flip                  |
|            | Directional up    | Size +                |
|            | Directional down  | Size -                |
|            | Trigger           | Grab                  |
|            | A button          | Group selected shapes |
